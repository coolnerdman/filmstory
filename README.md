# FilmStory

아날로그 필름 사진 애호가를 위한 웹 앱.

## 목표 (Target)
- 필름 사용 일지 및 가계부 기능.
- 필름 스캔본 저장 및 '밀착인화(Contact Sheet)' 스타일 뷰어.
- PC와 모바일 모두 지원 (Responsive Web App).

## 기술 스택
- Frontend: React (Vite) + Tailwind CSS
- Backend: (To be decided - Supabase recommended)
- Deployment: Vercel / Netlify

## 기능 명세
1. **필름 가계부 (Film Ledger)**:
    - 필름 구매, 현상, 스캔 비용 기록.
    - 월별/연도별 통계 그래프.
2. **디지털 밀착인화 (Contact Sheet)**:
    - 스캔 이미지 업로드.
    - 필름 종류, 촬영 기간 메타데이터 관리.
    - 갤러리 뷰어.
